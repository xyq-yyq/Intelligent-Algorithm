algorithm:
	ga_al.py:标准遗传算法（GA）。编码方式为二进制向量，长度等于候选哈希位总数 lp，且恰好包含 ld 个 1。采用锦标赛选择（tournament size = 3）选出亲代，以单点交叉生成子代，交叉后通过基数修复确保子代中 1 的个数恰好为 ld，随后以一定概率触发随机 1↔0 交换变异。
	iga_al.py:改进型遗传算法（IGA）。在标准遗传算法基础上集成了三项改进策略：1.禁忌搜索局部寻优，通过向量化方式系统性地探索单步交换邻域2.路径重连重组，利用精英池中的高质量解为亲代提供结构化重组方向，沿路径搜索更优解；3.确定性拥挤，采用小生境技术，让子代与最相似的亲代竞争，维持种群多样性。
	pmits_al.py:并行文化基因迭代禁忌搜索（PMITS），融合文化基因算法与禁忌搜索。算法在种群初始化后，对每个个体执行禁忌搜索获得初始局部最优。主循环中，每个个体通过偏斜交叉与随机选择的另一个个体交换部分基因，并以一定概率执行突变操作（随机交换若干个1和0），随后进行局部搜索将解提升至局部最优，再以禁忌搜索进一步强化解的质量。算法采用长期记忆机制缓存已访问解的禁忌搜索结果，避免重复计算；在种群中一定比例的个体收敛至相同解时提前终止；长时间无改进时触发重启机制重置种群。
	pso_adv_al.py:二进制粒子群优化（BPSO）。集成了三种改进机制：1.惯性权重衰减，从w_start 线性衰减至w_end，前期强探索后期强开发；2.重启策略，当全局最优连续多代无改进时，保留当前全局最优解，其余粒子完全随机重置，帮助跳出局部最优；3. 多样性增强，当粒子群与全局最优的平均海明距离低于阈值时，以全局最优为基准对部分粒子进行扰动（随机翻转若干位），维持种群多样性。
	random_al.py:随机选择基线。随机抽取 ld 个比特，不做任何优化。
hashing：
	DeepHN.py:深度哈希网络（DeepHN）的训练与编码。模型结构为单层全连接网络，将输入特征映射为nbits维输出。训练损失函数由两部分组成：1.成对相似度损失，基于标签构造相似度矩阵，约束相似样本的哈希码内积趋近于1，不相似样本趋近于-1；2.量化损失，通过 tanh(β·x) 近似二值化，鼓励输出值趋近于±1。训练完成后，以0为阈值将输出二值化为{0,1}哈希码。
	hash_pool.py:哈希池构建脚本，整合四种哈希方法生成统一的 288 位哈希池。流程如下：1.从本地文件夹加载CIFAR-10图像；2.使用预训练ResNet-50提取2048维图像特征；3.分别训练四种哈希方法（DeepHN、HashNet、IDeepHN、LSH），各生成72位哈希码；4.将四种哈希码按行拼接成形状为 (288, n) 的哈希池；5.保存为npz文件，同时保存特征和标签供后续计算亲和矩阵使用。
	hash_pool_2.py:构建 216 位传统哈希池：KLSH + PCAH + LSH（各 72 位）。整合三种经典哈希方法生成统一的 216 位哈希池。流程如下：1.尝试复用hash_pool.py中已提取的 ResNet-50 特征，若不存在则从零提取；2.分别生成三种传统哈希方法（KLSH、PCAH、LSH），各生成72位哈希码；3.将三种哈希码按行拼接成形状为 (216, n) 的哈希池；4. 保存为npz 文件。与 hash_pool.py 的区别在于，本脚本仅使用传统哈希方法，适用于对比实验中的传统基线方法。
	HashNet.py:HashNet的实现。采用连续化方法训练深度哈希网络，通过tanh函数逐步逼近sign函数，实现端到端的哈希码学习。
	IDeepHN.py:改进深度哈希网络（Improved Deep Hashing Network）的实现。在DeepHN基础上引入软相似度标签，使用更精细的成对相似度权重，提升哈希码的判别能力。
	ITQ.py:迭代量化哈希（Iterative Quantization）的实现。通过正交旋转矩阵对PCA投影结果进行迭代优化，最小化量化误差，提升哈希码质量。
	KLSH.py:核化局部敏感哈希（Kernelized Locality-Sensitive Hashing）的实现。在LSH基础上引入核函数，将数据映射到高维核空间后再进行随机投影，捕捉非线性结构。
	LSH.py:局部敏感哈希（Locality-Sensitive Hashing）的实现。通过随机投影将高维特征映射为二进制哈希码，是一种无监督哈希方法。
	PCAH.py:PCA 哈希（PCA Hashing）的实现。对数据进行PCA降维，取前nbits个主成分方向，以中位数为阈值进行二值化得到哈希码。
test:
	compare_methods.py:PMITS选择与四种单一哈希方法的对比实验。对比对象为：PMITS（从 288 位池中优化选出ld个比特）、DeepHN（仅使用前ld位）、HashNet（仅使用前ld位）、IDeepHN（仅使用前ld位）、LSH（仅使用前ld位）。PMITS运行50次取均值，单一哈希方法为确定性结果。输出折线对比图和柱状对比图。
	compare_methods_2.py:PMITS选择与三种传统哈希方法的对比实验。对比对象为：PMITS（从216位传统池中优化选出ld个比特）、KLSH（仅使用前ld位）、PCAH（仅使用前ld位）、LSH（仅使用前ld位）。输出折线对比图和柱状对比图。
compare.py:主对比脚本。五种智能算法（Random、GA、PMITS、IGA、BPSO）在多个 LD 下的性能对比，生成指标图、耗时箱线图、收敛曲线图。
config.py:全局配置文件。包含数据集路径、哈希池存储路径、CIFAR-10类别名称、训练/测试样本数量、批大小、每方法哈希位数、总哈希位数、待测试哈希码长度列表、亲和矩阵计算参数、随机种子（SEED）等。
utils.py:工具函数库。包含：相似度矩阵计算、拉普拉斯矩阵计算、哈希位可靠性计算、互信息矩阵计算、亲和矩阵计算、目标函数计算、检索指标计算等核心函数。
